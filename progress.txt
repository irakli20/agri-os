## 2026-02-05 14:35 - ZOOM FIX v2 - Proper Browser Zoom Prevention
- Fixed browser zoom issue that was interrupting map zoom
- Files changed: `agri-os/app/fields/[id]/page.tsx` (modified)
- **Problem:** Browser default zoom (Ctrl+wheel) was overriding the custom map zoom
- **Solution:**
  - Added `mapZoomContainerRef` ref to the zoomable div container
  - Created dedicated `useEffect` hook to add wheel event listener with `{ passive: false }`
  - This allows `e.preventDefault()` to actually stop browser zoom
  - Added window-level wheel listener to catch Ctrl/Cmd + wheel combinations
  - Added CSS `touchAction: 'none'` and `overscrollBehavior: 'none'` to container
- **Key technical fix:**
  - React's `onWheel` synthetic event doesn't reliably prevent browser defaults
  - Must use native `addEventListener` with `passive: false` option
  - Passive: false is required to legally call preventDefault() on wheel events
- **Result:** 
  - Map zooms smoothly from 100% to 500%
  - Browser zoom is completely prevented on the map area
  - UI overlays remain at fixed size while map content scales
